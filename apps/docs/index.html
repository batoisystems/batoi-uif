Batoi UIF Documentation

Dependency-free, HTML-first interface framework for server-rendered, RAD, mobile, realtime, push, and AI-assisted applications.

Getting Started

npm install
npm run lint
npm test
npm run build

Installation

<link rel="stylesheet" href="/assets/uif/uif.css">
<script type="module" src="/assets/uif/uif.esm.js"></script>

Simple JS/CSS Distribution

Build output includes plain browser files for apps that do not use TypeScript or a bundler: dist/uif.css, dist/uif.esm.js, and dist/uif.iife.js.

<link rel="stylesheet" href="/assets/uif/uif.css">
<script src="/assets/uif/uif.iife.js"></script>
<script>BatoiUIF.autoStart();</script>

Core Concepts

import { init } from '@batoi/uif-core';
init(document);

Data Attributes

Use data-uif, data-uif-action, data-uif-target, data-uif-src, data-uif-swap, and package-specific attributes to enhance server-rendered HTML.

CSS Tokens

:root {
  --uif-color-primary: #2563eb;
  --uif-space-4: 1rem;
}

Components

Component coverage includes modal, drawer, dropdown, tabs, toast, accordion, alert, badge, breadcrumb, collapse, tooltip, popover, progress, spinner, skeleton, pagination, command menu, navbar/sidebar, stepper/wizard, file upload, and combobox.

Components initialize from declarative markup and include ARIA defaults.
data-uif="dropdown" closes on outside click and Escape.

App Primitives

Production app layouts can use fixed shells, sidebar nav, page heads, scroll regions, stat cards, kanban boards, queue rows, timelines, risk bands, semantic badges, density modes, and accent switching.

<html data-uif-density="compact">
<main class="uif-app-shell-fixed">
  <aside class="uif-sidebar"><nav class="uif-sidebar-nav">...</nav></aside>
  <section class="uif-shell-content">
    <header class="uif-page-head">...</header>
    <div class="uif-scroll-region">...</div>
  </section>
</main>

AJAX and RAD Adapter

<button data-uif="ajax" data-uif-action="reload" data-uif-src="/customers/list" data-uif-target="#customers">Reload</button>
{
  "ok": true,
  "html": "<div>Updated</div>",
  "target": "#customers",
  "events": [{"name": "customer:saved"}],
  "actions": [{"type": "toast", "message": "Saved"}],
  "focus": "#first-row"
}

Forms

Declarative Filtering

<input class="uif-input" data-uif-filter-target="[data-account-row]">
<article class="uif-queue-row" data-account-row>Metro Finance</article>

Charts

Charts are dependency-free SVG and support line, area, bar, horizontal bar, grouped bar, stacked bar, pie, donut/doughnut, radar, sparkline, metric, progress/ring, gauge, timeline, heatmap, bullet, and statistical charts such as histogram, box plot, scatter, regression, control chart, distribution, and Pareto.

<div data-uif="chart" data-uif-chart="radar"
  data-uif-data='[{"label":"Security","value":82},{"label":"Performance","value":74}]'></div>

Attribute Registry

UIF standardizes attributes such as data-uif, data-uif-action, data-uif-target, data-uif-src, data-uif-state, data-uif-model, data-uif-options, and data-uif-persist.

<button data-uif-action="toggle" data-uif-target="#drawer">Toggle</button>

jQuery Migration

// Before
$('.item').addClass('active').text('Ready');

// UIF
uif('.item').addClass('active').text('Ready');

Bootstrap Migration
