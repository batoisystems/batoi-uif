pnpm install

Core Concepts

Initialize once, enhance progressively, and emit lifecycle events from plain DOM nodes.

<script type="module">
  import { init } from '@batoi/uif-core';
  init(document);
</script>

Data Attributes

UIF behavior is declared with attributes such as data-uif, data-uif-action, and data-uif-target.

<button data-uif-action="toggle" data-uif-target="#drawer">Menu</button>

Components

Tabs use ARIA roles and arrow-key navigation.
data-uif="tabs"

AJAX and RAD Adapter

<button data-uif="ajax" data-uif-action="reload" data-uif-src="/customers/list" data-uif-target="#customers">
  Reload customers
</button>
<div id="customers"></div>
